📦GitHub 文件加速

✨ 支持带协议头(https://)或不带的 GitHub 链接，直接回车或点击右侧按钮即可加速下载

🚀 支持 release、archive、raw、blob、gist 等资源类型

⚠️ 注意：暂不支持文件夹下载，本站仅供演示，建议自建

📃 合法输入示例：

📄 分支源码：github.com/username/project/archive/master.zip

📁 release源码：github.com/username/project/archive/v0.1.0.tar.gz

📂 release文件：github.com/username/project/releases/download/v0.1.0/example.zip

💾 commit文件：github.com/username/project/blob/1111111111111111111111111111/filename

🖨️ gist：gist.githubusercontent.com/username/id/raw/file.py
